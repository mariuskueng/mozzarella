## Usage

## Get it

```
$ git clone git@github.com:mariuskueng/mozzarella.git
```

### Install Meteor if you haven't already

```
$ curl https://install.meteor.com/ | sh
```

### Run it

```
$ cd mozzarella
$ meteor
```
Open `http://localhost:3000` in your browser.
